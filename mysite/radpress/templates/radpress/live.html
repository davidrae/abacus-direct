Live Monitoring

Abacus has incredibly fast start up speeds and one of the fastest analytics engines on the market. With its intelligence, Abacus delivers a self-aware system which can, for example, reboot itself and repair software problems in cameras and other intelligent devices.

Abacus uses advanced counting algorithms, to deliver accurate and reliable information, quickly and effectively. By connecting to external devices, Abacus can utilize the functionality of smart devices and other counting systems.

  • View live camera feeds from the dashboard and see events as they happen. View individual cameras, or select specific models to receive bespoke data.

  • Live monitoring of data presents the user with counting data in real time. They can view events as they happen, to make appropriate managerial decisions quickly and efficiently.

  • Abacus offers the user the ability to program their own dashboard parameters. This means they can see calculated figures in real time. A user can find out the total number of free spaces in a car park, or a percentage of the capacity filled for example.
