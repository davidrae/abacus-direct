Reports

The main purpose of the reporting feature is to display accurate, historical counting data in a format that is easily read by the user.

Search

Abacus uses advanced, searching algorithms to search for the data you need from the database. It delivers the data quickly and efficiently.

Customize

Use a variety of filters to deliver bespoke, custom data from a plethora of sources. Use this to fetch data from different sites, and compare statistics based on different parameters.

Time Range

Abacus can further filter data based on the time you want to see data from. Fetch data on hourly, daily, weekly, monthly or even yearly figures.

Dynamic time ranges can be created to further enhance the comparison of your data. Compare statistics from the previous week or month, and create predictions for the future.

Event Types

Search by a variety of event rules, or types. Find figures such as in/out count, loiter time, object size, speed, enter/exit count, object appearances/disappearances etc.

Chart Types

Display the data in a format you deem appropriate. Line and spline graphs can be used to compare multiple statistical series. Area charts can be used to show trends over time. Bar graphs can be used to compare different groups of data. Pie charts can be used to compare precentage data. And matrices are used to show raw data in a table format.

Programmable Views

Program your own filters to lay on top of the reports. Have your own set of custom reports ready for your viewing every time you log in.

Export

Export the report data in a variety of formats. When a report is created, you can send it to other users, email it, print it, create a PDF, or create a CSV file for use within Excel.
