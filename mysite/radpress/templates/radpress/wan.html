Wide Area Network

Abacus was designed to be modular. The many pieces that make up the system can be moved around, put into separate systems, or even run on their own entirely. This functionality has allowed Abacus to link with other installations and harvest counts that it may not have been accessible to before.

As an example, there can be a main Abacus installation at a head office, with several, smaller systems off site that harvest information locally, before sending it on to the main Abacus server. The main server installation can view data from each and every site that it receives counts from.

This solution was created with international businesses in mind. If you have several sites spanning a wide area, this solution will allow you to view the data from all your sites from one application.
